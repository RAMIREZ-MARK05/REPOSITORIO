Kathetensatz und Satz des Pythagoras

Die folgende HTML5-App zeigt ein rechtwinkliges Dreieck, bei dem der Scheitel des rechten Winkels durch Ziehen mit der Maus verändert werden kann.

HTML5-Canvas nicht unterstützt!
Kathetensatz:
Im rechtwinkligen Dreieck ist das Quadrat über einer Kathete flächengleich zum Rechteck aus der Hypotenuse und dem anliegenden Hypotenusenabschnitt.
a2 = c p
b2 = c q
a2 + b2 = c p + c q = c (p + q) = c · c = c2
Satz des Pythagoras:
Im rechtwinkligen Dreieck haben die beiden Kathetenquadrate zusammen die gleiche Fläche wie das Hypotenusenquadrat.
a2 + b2 = c2
